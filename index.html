🔧 Connect Firebase

Vinkle needs a Firebase Realtime Database to work. Set it up in just 4 steps!

1

Go to console.firebase.google.com → Add project → name it → Create

2

Build → Realtime Database → Create Database → choose region → test mode → Enable

3

⚙️ Project Settings → Your apps → Web → register → copy firebaseConfig

4

Open vinkle.html in a text editor → find FIREBASE_CONFIG → paste values → reload
