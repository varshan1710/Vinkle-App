Circles uses Firebase Realtime Database as its backend so anyone in the world can join your circles in real time — all messages are encrypted in your browser before being sent.

Follow these quick steps to connect your free Firebase project:

1

Go to console.firebase.google.com → Add project → name it anything → disable Analytics → Create project

2

In your project: Build → Realtime Database → Create Database → choose any region → start in test mode → Enable

3

Click the ⚙️ gear icon → Project Settings → Your apps → </> Web → register the app → copy the firebaseConfig object

4

Open index.html in a text editor → scroll to the bottom → find FIREBASE_CONFIG → replace the placeholder values with your config values, then reload

// Replace these values in index.html: const FIREBASE_CONFIG = { apiKey: "AIzaSy...", authDomain: "my-circles.firebaseapp.com", databaseURL: "https://my-circles-default-rtdb.firebaseio.com", projectId: "my-circles", storageBucket: "my-circles.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc..." };
